# (async) getAuthorizeUrl(props, sdkConfig) → {Promise.<getAuthorizeUrlResponse>}

Authorize a new user In order to write or read data from digi.me, we first need to create an access token. Access tokens are linked to a contract, and it is possible to create multiple access tokens that access to the same digi.me libary. This function is called when:

  • Authorize a new user. You have the option to also onboard a service during this process.
  • An existing user authorizing a new contract. Existing user’s refresh token has expired and we need to extend it.
